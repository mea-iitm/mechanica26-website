# **App Name**: Mechanica Nexus

## Core Features:

- Dynamic Festival Content Pages: Engaging presentation of the Home, About, Summit, Contact, Mission, Vision, and Why Us sections with interactive elements such as speaker profiles and clickable mission cards.
- Event & Workshop Listing: Comprehensive showcase of all technical events and workshops, featuring detailed descriptions, dates, and 'register' call-to-action links in an organized card layout.
- Interactive Schedule Timeline: A vertical, animated timeline displaying the full 3-day festival schedule with animated connecting lines and circular timeline nodes for clear event flow.
- Personalized Mechanica Passport: A digital passport system, visually resembling a modern passport card, allowing participants to track event attendance, collect activity stamps/badges, and view their overall engagement. This feature requires basic data storage for user participation.
- Festival Shop (Passes & Merchandise): A dedicated section for browsing and purchasing event passes and official Mechanica-branded merchandise, complete with images, descriptions, prices, and buy buttons.
- Interactive Stats Display: Engaging 3D interactive tiles with hover depth animations and subtle glow effects to highlight key festival statistics such as student count, teams, and participation numbers.
- AI Personalized Itinerary Tool: A generative AI tool integrated into the passport system to assist registered participants in creating tailored festival itineraries based on their selected interests and the event schedule.

## Style Guidelines:

- Light Scheme Background: A very soft grey-lavender, complementing the primary palette, HSL(247, 15%, 95%) which converts to #ECEEF5. This provides a clean canvas for content.
- Dark Scheme Background: A deep charcoal tone for the dark theme, HSL(247, 15%, 15%) which converts to #212128. This offers a sleek, futuristic backdrop while maintaining visual hierarchy.
- Primary Color (Deep Purple): Used for key elements and brand identity, HSL(247, 100%, 71%) which converts to #6A5CFF. This bold and vibrant choice aligns with the technical and futuristic aesthetic requested.
- Accent Color (Deep Cyan-Blue): An analogous deep blue, used to highlight interactive elements and provide contrast, HSL(217, 90%, 35%) which converts to #0A5785. This ensures significant difference in brightness and saturation from the primary color for effective visual hierarchy. The user-specified 'Electric Blue' (#3DA9FC) will be utilized within glow effects and secondary interactive accents to create dynamic 'glowing purple-blue gradients' for emphasis.
- Headline Font: 'Space Grotesk' (sans-serif), chosen for its modern, techy, and slightly computerized aesthetic, perfectly fitting the futuristic engineering theme. This font is strong and legible for titles.
- Body Font: 'Inter' (sans-serif), selected for its clear readability, neutral modern look, and versatility, ensuring that longer text blocks are comfortable to read in both light and dark themes.
- Use minimalist, vector-based line icons with subtle glow effects and a sleek finish to reinforce the futuristic and high-tech engineering aesthetic throughout the application. Social media icons should also feature distinct hover glows.
- A clean, modular, and responsive grid-based layout with generous negative space to enhance content focus. Prominent use of glassmorphism cards, interactive 3D tiles for statistics, and a distinctive 'realistic digital passport card' UI will define the user experience. Content should be clearly separated and logically grouped for intuitive navigation across all screen sizes.
- Implement smooth and responsive hover transitions on interactive elements, subtle scroll animations between content sections, and elegant soft shadows for depth. Key elements like cards and the 3D stat tiles will feature noticeable glow effects and hover depth animations. The schedule timeline will incorporate animated connecting lines and circular nodes for dynamic progression.
